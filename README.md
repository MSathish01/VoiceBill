### PDF Generation

2. Add items via voice or manual entry
3. Click "Download PDF" for professional bill

1. Click "Choose PDF File" in the PDF Upload section
2. Select a previously generated bill PDF
3. Items are automatically extracted and populated
4. Edit interface opens for immediate editing
5. Make changes as needed
6. Generate new PDF with updated items

**Supported PDF Sources:**
- Bills generated by this application
- PDFs with similar table structure (S.No, Item, Qty, Rate, Total)

## ğŸ—ï¸ Technical Architecture

### Frontend Stack
- **React 18** - Modern UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **CSS3** - Responsive design with flexbox/grid

### Core Modules
- **voiceParser.ts** - Universal lexicon and parsing engine
- **tamilLinguisticEngine.ts** - Tamil language processing
- **pdfGeneratorV2.ts** - HTML-to-canvas PDF generation
- **pdfExtractor.ts** - PDF parsing and item extraction
- **VoiceControls.tsx** - Voice recognition interface
- **CustomerForm.tsx** - Customer details management
- **PdfUpload.tsx** - PDF upload and extraction UI

### Voice Processing Pipeline
1. **Speech Recognition** - Web Speech API
2. **ASR Preprocessing** - Remove artifacts, handle code-mixing
3. **Semantic Verification** - Fuzzy matching against lexicon
4. **Component Extraction** - Parse item, quantity, price
5. **Auto-commit** - Move completed items to bill
